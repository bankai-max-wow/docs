---
title: "List Models"
api: "GET /models"
description: "Returns a list of all AI models currently available on Samurai AI."
---

<RequestExample>
```bash cURL
curl https://www.samuraiapi.in/v1/models \
  -H "Authorization: Bearer $SAMURAI_API_KEY"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-samurai-YOUR_KEY",
    base_url="https://www.samuraiapi.in/v1"
)

models = client.models.list()
for model in models.data:
    print(model.id)
```

```javascript Node.js
const models = await client.models.list();
models.data.forEach(m => console.log(m.id));
```
</RequestExample>
