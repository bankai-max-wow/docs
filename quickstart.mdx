## Step 1: Get your API Key

1. Sign up at [samurai-ai.com](https://samurai-ai.com)
2. Go to your [Dashboard](https://samurai-ai.com/dashboard)
3. Click **API Keys** → **Create New Key**
4. Copy your key — it starts with `sk-samurai-`

<Warning>
  Never share your API key or commit it to version control. Always use environment variables.
</Warning>

## Step 2: Install the SDK

Samurai AI is 100% OpenAI-compatible. Use the official OpenAI SDK — no special package needed:

<CodeGroup>
```bash pip
pip install openai
```
```bash npm
npm install openai
```
```bash yarn
yarn add openai
```
</CodeGroup>

## Step 3: Make your first request

<CodeGroup>
```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-samurai-YOUR_KEY",
    base_url="https://api.samurai-ai.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello! What can you do?"}]
)

print(response.choices[0].message.content)
```
```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-samurai-YOUR_KEY',
  baseURL: 'https://api.samurai-ai.com/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello! What can you do?' }]
});

console.log(response.choices[0].message.content);
```
```bash cURL
curl https://api.samurai-ai.com/v1/chat/completions \
  -H "Authorization: Bearer sk-samurai-YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello! What can you do?"}]
  }'
```
</CodeGroup>
