## One-Line Migration

<CodeGroup>
```python Python — Before (OpenAI)
from openai import OpenAI
client = OpenAI(api_key="sk-openai-...")
```
```python Python — After (Samurai AI)
from openai import OpenAI
client = OpenAI(
    api_key="sk-samurai-YOUR_KEY",
    base_url="https://samurai-ai.com/v1"   # Only change
)
```
</CodeGroup>

<CodeGroup>
```javascript Node.js — Before (OpenAI)
const client = new OpenAI({ apiKey: 'sk-openai-...' });
```
```javascript Node.js — After (Samurai AI)
const client = new OpenAI({
  apiKey: 'sk-samurai-YOUR_KEY',
  baseURL: 'https://samurai-ai.com/v1'    // Only change
});
```
</CodeGroup>

## Compatible Libraries

| Library | Language | How to Configure |
|---------|----------|-----------------|
| `openai` | Python / Node.js | `base_url` / `baseURL` |
| `langchain` | Python / JS | `openai_api_base` |
| `llama_index` | Python | `api_base` |
| `@ai-sdk/openai` | TypeScript | `baseURL` in `createOpenAI()` |
| `litellm` | Python | `api_base` param |

<Note>
  No special SDK needed. Any library that supports a custom `base_url` works instantly.
</Note>
